SkillBridge

SkillBridge is a responsive React-based job & internship portal designed to connect students, professionals, and employers on one unified platform. It allows users to explore job listings, apply for roles, and manage their applications through an intuitive dashboard.

 Features

 Job Listings — Browse latest internships & job opportunities

 Company Profiles — Explore verified companies & recruiters

 Apply System — Simple and fast application submission

 Dashboard — Personalized dashboard for tracking applications

 Responsive UI — Works seamlessly on desktop & mobile

 Built with Vite + React — Super-fast and optimized build setup
